# Project Context

## Goals
- Maintain source-of-truth OpenAPI 3.1 specifications for RegistryAccord.
- Ensure high quality via linting (Spectral) and strict style guides.

## Architecture
- **Specs**: `openapi/{service}/{version}/openapi.yaml`
- **Common**: `openapi/common/` (Shared schemas)
- **Docs**: `docs/requirements/INDEX.md`
