# Edge Case Examples

This directory contains examples for boundary conditions and unusual scenarios.

## Purpose

- Demonstrate handling of edge cases
- Show pagination boundaries
- Illustrate empty responses
- Provide unicode/internationalization examples

## Files

- `empty-list-response.json` - Empty collection response
- `pagination-end.json` - Last page of paginated results
- `maximum-page-size.json` - Request exceeding max page size
- `unicode-content.json` - Content with emoji and multi-byte characters
- `tombstone-deleted-resource.json` - Response for soft-deleted content
